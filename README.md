# DRS Karting website
 
Frontend layout project for a karting center website.
 
## Overview
 
DRS Karting is a static website project created as part of frontend practice.  
The goal of this project was to build a clean layout using core web technologies.
 
This project focuses on working with HTML, CSS, and JavaScript to create a functional website structure.
 
## Tech Stack
 
- HTML5
- CSS3
- JavaScript (Vanilla)

## Getting Started
 
To run the project locally:
 
1. Clone the repository:
```bash
git clone https://github.com/artem-maslov-dev/drs-karting-website.git
```
 
2. Open the project folder
3. Run `index.html` in your browser

No build tools or dependencies are required.
